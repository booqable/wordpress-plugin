Let your customers directly make rental reservations from your website.

== Description ==

Make sure to sign up for an account at https://booqable.com first.
After you've added the products you wan't to rent out you can install the plugin.

== Installation ==

1. Upload the `booqable` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup your company name in the booqable plugin settings
